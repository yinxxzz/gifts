# 6月赠品图片 · 放这里

把素材放进**本文件夹**，文件名与品名一致（支持 `.jpg` `.jpeg` `.png` `.webp`）。

## 命名规则

| 用途 | 文件名示例 |
|------|------------|
| 页面主图 / 下载 zip | `斑马书包.jpg` |
| 同品多图（备选、素材库） | `斑马书包-02.jpg`、`斑马书包-03.jpg` … 两位数序号 |
| 详情长图（A7 / PC 拼接） | `词典笔-详情长图.jpg` 或 `语数英点读书-07.jpg` |
| 视频素材 | `斑马书包-视频.mov` |

- 主图放期次根目录（如 `assets/2026-06/斑马书包.jpg`），JSON 填同路径。
- 同品多张时用 `-02`、`-03` 递增；`-01` 可省略，主图直接用品名。
- 避免 `DSC09453.jpg` 这类相机原名，不利于页面自动匹配。
