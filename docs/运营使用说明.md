# 赠品策略页 · 运营上新说明

> 销售：驿站 → **赠品策略**，打开就看，不用改。

---

## 第 0 步：准备工作

页面最底部 **···** 三击 → 下载 **Cursor Skill**，解压后放到本机 `~/.cursor/skills/sales-gift-strategy-ops/`。

Skill 里写了全流程（含新电脑怎么装、怎么预览、怎么上线），在 Cursor 里打开 `sales-gift-strategy/` 文件夹即可开始。

---

## 第一步：准备材料，发给 Cursor

### 1. 本期赠品清单（石墨、文字、截图均可）

说明本期卖什么、怎么选，例如：

```
7月系统课加赠
- 主礼包：4选1，品有：斑马书包、语数英点读书、词典笔、汉语拼音学习机
- 副礼包：6选2，品有：仿真早教机、小风扇、得力地球仪、科学绘本、恐龙翻翻书、太空翻翻书
- 主推：主礼包推汉语拼音学习机；副礼包推恐龙翻翻书 + 仿真早教机
- 新品：汉语拼音学习机、仿真早教机、小风扇、恐龙翻翻书、太空翻翻书
```


### 3. 图片素材

- 图片整理好后告诉Cursor路径位置

---

## 第三步：预览确认

检查：期次、日期、策略介绍、图、卖点、主推标签、期次状态。

页面上 **三击** 可微调文案（只改本机；定稿以文件为准）。

---

## 第四步：上线

预览 OK 后，在 Cursor 里说：「帮我把 sales-gift-strategy 部署上线」。  
Cursor 会按 Skill / `env.md` 上传到 Run 平台并重启静态服务。
